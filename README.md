# Sato Clinic

さとうクリニックの紹介サイトです。  
HTML と CSS だけで構成された静的サイトで、画像素材は `img/` にまとめています。

## 特徴

- トップページ
- お知らせ
- 院長あいさつ
- 診療案内
- 当院について
- スタッフ紹介
- アクセス
- 診療時間

## ファイル構成

- `index.html` - 画面全体のHTML
- `css/style.css` - レイアウトとデザイン
- `img/` - ロゴ、写真、イラストなどの画像素材
- `LICENSE` - ライセンス情報

## 開き方

1. `index.html` をブラウザで開きます。
2. 画像やレイアウトが表示されることを確認します。

## 補足

- このサイトは静的ページです。
- 文字や画像を変更する場合は、`index.html` と `css/style.css`、`img/` の内容を編集してください。
