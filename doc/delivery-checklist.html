納品一覧・確認書

納品対象となるソース一式、補助ドキュメント、および引き継ぎ時の確認事項を整理しています。

1. 納品物一覧

区分 内容
ページHTMLトップ、詳細、一覧、問い合わせ、お知らせ更新導線を含むHTML一式
CSScss/style.css
JavaScriptjs/news-data.js、js/news.js、js/contact.js、js/news-update-popup.js
画像img/ 配下の素材一式
文書README.md、AGENTS.md、doc/ 配下一式

2. 既知の仕様

  • お知らせは静的データ管理であり、管理画面投稿機能は未実装
  • お知らせ更新導線は専用画面へ遷移するが、機能本体は `Comming Soon` 表示
  • 電話、メール導線の一部もモックとして `Comming Soon` ポップアップを表示
  • フォーム送信は実メール送信ではなく起票確認用のモック

3. 運用上の注意

  • お知らせ内容を更新する場合は js/news-data.js を編集する
  • 画像追加時はスマホ表示の表示速度に注意する
  • HTML保存時は UTF-8 を維持する
  • ヘッダ導線やページアンカー変更時は全ページの整合性を確認する

4. 受領確認事項

  • ソース一式が欠けなく格納されていること
  • 納品ドキュメントが UTF-8 で閲覧できること
  • 主要ページがブラウザで表示できること
  • GitHub Pages 公開に必要な静的ファイルのみで構成されていること
